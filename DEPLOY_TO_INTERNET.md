# 🌐 인터넷에 올려서 태블릿에서 바로 접속하기

## ⚡ 가장 빠른 방법: Netlify Drop (30초만에 완료!)

### 단계별 가이드

1. **파일 준비**
   - `index_standalone.html` 파일을 찾아서 이름을 `index.html`로 변경
   - (파일을 우클릭 → 이름 바꾸기)

2. **Netlify Drop 접속**
   - 브라우저에서 https://app.netlify.com/drop 열기
   - (회원가입 필요 없음!)

3. **파일 업로드**
   - 이름을 변경한 `index.html` 파일을 드래그 앤 드롭
   - 또는 "browse to upload" 클릭해서 파일 선택

4. **완료!**
   - 즉시 접속 가능한 링크가 생성됩니다!
   - 예: `https://amazing-name-12345.netlify.app`
   - 이 링크를 태블릿 브라우저에서 바로 사용 가능!

5. **태블릿에서 접속**
   - 태블릿 브라우저 주소창에 생성된 링크 입력
   - 바로 Safe Door 웹사이트 접속 가능!

## 📚 다른 방법들

### GitHub Pages (무료, 영구적)

1. GitHub 계정 만들기: https://github.com/signup
2. 새 저장소 만들기: https://github.com/new
   - 저장소 이름: `safe-door` (원하는 이름)
   - Public 선택
3. 파일 업로드:
   - `index_standalone.html` → `index.html`로 이름 변경
   - 저장소 페이지에서 "uploading an existing file" 클릭
   - 파일 드래그 앤 드롭 후 Commit
4. Pages 활성화:
   - Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Save
5. 접속: `https://[사용자이름].github.io/[저장소이름]/`

---

### Vercel (빠르고 안정적)

1. https://vercel.com 접속 (GitHub 로그인 가능)
2. "Add New..." → "Project"
3. GitHub 저장소 연결 또는 파일 업로드
4. 자동 배포 및 링크 제공
