【 주요 원인 1 】파일을 직접 더블클릭해서 열었을 때
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
문제: file:// 프로토콜로 열면 브라우저 보안 정책으로 기능 제한됨
해결: 웹 서버로 실행하거나 GitHub Pages에 배포

  → start.bat 실행
  → 또는 start_tablet.bat 실행
  → 또는 GitHub Pages 배포 (GITHUB_배포_가이드.md 참고)

【 주요 원인 2 】파일 인코딩 문제
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
문제: 한글이 깨져서 표시됨
해결: 파일을 UTF-8 인코딩으로 저장

  → 메모장 또는 VS Code로 파일 열기
  → 파일 → 다른 이름으로 저장 → 인코딩: UTF-8 선택

【 주요 원인 3 】브라우저 위치 권한
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
문제: 지도 기능이 작동하지 않음
해결: 브라우저에서 위치 권한 허용

  → 주소창 왼쪽 자물쇠 아이콘 클릭
  → 위치 → 허용 선택

【 주요 원인 4 】브라우저 호환성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
문제: 오래된 브라우저에서는 작동하지 않음
해결: 최신 브라우저 사용

  → Chrome, Edge, Firefox 최신 버전 사용
  → Internet Explorer는 사용하지 마세요

【 주요 원인 5 】JavaScript 비활성화
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
문제: 페이지가 전혀 작동하지 않음
해결: JavaScript 활성화

  → 브라우저 설정 → 사이트 설정 → JavaScript → 허용
