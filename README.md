# Safe Door 앱

안전장구 착용 확인 시스템 웹 애플리케이션입니다.

## 기능

- 안전장구 착용 확인 (메인 페이지) - 이미지 6번이 초기 화면
- 출입기록 조회
- 관리자 로그인
- 대시보드 (지도 및 통계)
- 안전장구 착용 결과 메시지

## 설치 및 실행

### 필수 요구사항
- Node.js (v16 이상) 및 npm

### 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev
```

개발 서버가 시작되면 브라우저에서 표시된 주소(보통 http://localhost:5173)로 접속합니다.

### 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 사용 방법

1. 메인 페이지(초기 화면)에서 "확인" 버튼을 클릭하여 안전장구 착용을 확인합니다.
2. 관리자 비밀번호는 `admin123`입니다.
3. 각 화면은 자동으로 전환되거나 버튼을 통해 이동할 수 있습니다.

## 태블릿 최적화

이 웹사이트는 태블릿 환경에 최적화되어 있습니다:
- 터치 친화적인 버튼 크기 (최소 48px~64px)
- 태블릿 해상도에 맞는 폰트 크기
- 가로/세로 방향 지원
- 반응형 레이아웃
